Javascript

Javascript merupakan bahasa pemrogaman high-level , scripting , untype dan interpreted yang digunakan untuk membuat website agar terlihat interaktif dan dinamis.

Deklarasi Variabel

Adalah proses membuat variabel untuk menyimpan sebuah data.

Var
  • Dapat di deklarasi ulang di scope yang sama
  • Dapat di deklarasikan terlebih dahulu tanpa di assign value
  • Bisa di reassign
  • Support Hoisting
  • Dapat di akses dari Global Scope
  • Tidak dapat di akses jika berada didalam function scope
  • Dapat diakses ketika berada didalam block scope.
Let
  • Tidak bisa di deklarasi ulang di scope yang sama
  • Dapat di deklarasikan terlebih dahulu tanpa di assign value
  • Bisa di reassign
  • Tidak support Hoisting
  • Dapat di akses dari Global Scope
  • Tidak dapat di akses jika berada didalam function scope
  • Tidak dapat diakses ketika berada didalam block scope.
Const
  • Tidak bisa di deklarasi ulang di scope yang sama
  • Tidak dapat di deklarasikan terlebih dahulu tanpa di assign value
  • Tidak bisa di reassign
  • Tidak support Hoisting
  • Dapat di akses dari Global Scope
  • Tidak dapat di akses jika berada didalam function scope
  • Tidak dapat diakses ketika berada didalam block scope.

Scoping

Suatu proses menentukan dimana code kita baik variabel , function , objek diletakan dan di dapat akses.

Hoisting

Merupakan cara menggunakan atau mengakses variabel sebelum di deklarasikan dan ini hanya berlaku di variabel yang di buat menggunakan var.

Primitive Value

Didalam javascript terdapat dua tipe value pada javascript

Primitive value

Merupakan unit pemrosesan terkecil dan elemen yang paling sederhana yang tersedia didalam bahasa pemrograman. Primitive value terdiri dari :

  • Boolean
  • String
  • Bisa di reassign
  • Number
  • BigInt (ES11)
  • Undefined
  • Null
  • Symbol (ES6)

Refrence Value

Javascript menyediakan tiga tipe refrence value yaitu Object , array , dan function. Salah satu perbedaan antara Primitive value & Refrence value yaitu , ketika di copy primitive value akan membuat copyan dari variabel yang di copy , Sedangkan ketika refrence value di copy maka akan membuat refrence ke object yang sama dengan variabel yang di copy.

Destructuring

Destructuring merupakan ekspersi javascript yang memungkinkan kita untuk membuat salinan dari nilai array atau object ke variabel lain.

Spread Syntax

Merupakan syntax javascript yang berfungsi untuk membongkar atau menyebarkan isi dari object atau array

Funtion & Method

Function merupakan sekumpulan code yang bertugas untuk menjalankan sebuah instruksi tertentu yang dapat digunakan berulang ulang . Method adalah sebuah untuk sebuah function yang didefinisikan didalam sebuah object.

Control Flow

Control flow merupakan cara untur mengatur alur eksekusi pada statement atau jalannya program sesuai dengan keinginan kita. Control flow dapat diterapkan dengan beberapa cara :

Perulangan
  • For
  • while
  • do while
Pengkondisian
  • if else
  • switch case
  • Try & Catch
  • Continue
  • Throw

Synchronus & Asynchronus

Synchronus merupakan cara pengeksekusian program secara berurutan dan ouput yang dikeluarkan juga berurutan.

Asynchronus merupakan cara pengeksekusian program yang tidak berurutan , program akan melakukan eksekusi dan menampilkan output berdasarkan urutan eksekusi tercepat .

Callback

Callback sebutan untuk sebuah function yang dipassing sebagai argument ke sebuah function lain , dan callback akan di jalankan ketika function lain selesai di eksekusi.

Promise

Promise merupakan object javascript yang merepresentasikan keberhasilan atau kegagalan sebuah operasi asynchronus , dan promise mereturn value sesuai janji yang telah dibuat.

Method pada promise ada tiga diantaranya :

  • then()method ini akan di panggil ketika promise terpenuhi (resolve)
  • catch()method ini akan di panggil ketika promise gagal dipenuhi (reject)
  • catch()method ini akan di panggil ketika promise gagal dipenuhi (reject)

Asynchronus Function

Merupakan sebuah function yang di definisikan dengan keyword async pada function tersebut dan mereturn value berupa promise. Di dalam Asynchronus function berisi statement yang bersifat asynchronus , sehingga membutuhkan waktu untuk menyelesaikan proses dari statement tersebut.
