Bootstrap (CSS Framework)

Bootstrap merupakan salah satu css framework , dimana framework itu merupakan sekumpulan kode yang sudah dirangkai untuk menjalankan sebuah instruksi tertentu dengan tujuan untuk mempermudah kita dalam melakukan suatu pekerjaan seperti membuat tampilan atau melakukan tugas lainnya.

Container

Container merupakan konsep dasar yang dapat digunakan untuk membangun block pada bootstrap , container sering dijadikan sebagai pembungkus content di dalamnya , karena container yang dimiliki bootstrap ini bersifat responsif terhadap device kita.

Container tersebut juga memiliki konsep tersendiri yaitu seperti grid , didalam container kita bisa menentukan kolom dan baris untuk mengatur tata letak konten kita.

Navigation Bar

Bootstrap memiliki salah satu component yang siap dipakai , yaitu navigation-bar atau "navbar" , boostrap menyediakan berbagai macam variasi navbar seperti ada dropdown di dalamnya dan juga sudah responsif terhadap semua device kita .

Perbedaan Container & Container Fluid

Terdapat class yang dapat di definisikan ke element kita untuk memulai menggunakan container bootstrap , yaitu .container & container-fluid. Diantara container & container-fluid memiliki perbedaan yang simple saja yaitu terletak pada ukuran container.

Jika kita menggunakan container .container maka ukuran container tersebut akan sesuai dengan breakpoint yang telah ditetapkan oleh bootstrap

Jika kita menggunakan container .container-fluid maka ukuran container tersebut akan sesuai dengan breakpoint yang telah ditetapkan oleh bootstrap
